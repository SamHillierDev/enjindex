ENJindex, a trusted token explorer.

Final Year Project (FYP) for the Computer Science 2024 academic year at [Nottingham Trent University](https://ntu.ac.uk) (NTU).

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v20.6.1
```
```bash
npm -v
# 10.1.0
```

## Installation

Run the commands:
```bash
cd frontend
```
```bash
npm install
```
```bash
npm run dev
```
