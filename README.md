Final Year Project (FYP) for the Computer Science academic year 2023/24 at [Nottingham Trent University](https://www.ntu.ac.uk/course/computer-science).

## Requirements

Make sure you have Node and NPM installed::

```bash
node -v
# v20.6.1
```
```bash
npm -v
# 10.1.0
```

## Installation

Run the commands:
```bash
cd frontend
```
```bash
npm install
```
```bash
npm run dev
```
